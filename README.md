---

---

## ⚙️ Features

- Generate QR codes from any URL or text  
- Add a **background image** (PNG, JPG, JPEG)  
- Add a **background GIF** for animated QR codes  
- Automatically chooses `.png` or `.gif` output based on background  
- Preview QR codes in the app  
- Download the generated QR code  
- Adjustable QR scale for larger or smaller codes  
- Works both **with background** or **plain QR**
