# Animated QR Code Generator

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://generate-animated-qr-code.streamlit.app)

🎨 Generate QR codes with optional **image or GIF backgrounds**, fully interactive and downloadable.

---

## 🔗 Live App

You can try the live app here: [Animated QR Code Generator](https://generate-animated-qr-code.streamlit.app)

---

## 📦 Project Structure

├── app.py # Streamlit app
├── requirements.txt # Python dependencies
├── qr code generation.ipynb # Jupyter Notebook demo: generate QR codes with/without backgrounds
├── README.md # Project documentation
└── ANIMATED QR CODE GENERATOR.gif (optional header/logo)



---

## ⚙️ Features

- Generate QR codes from any URL or text  
- Add a **background image** (PNG, JPG, JPEG)  
- Add a **background GIF** for animated QR codes  
- Automatically chooses `.png` or `.gif` output based on background  
- Preview QR codes in the app  
- Download the generated QR code  
- Adjustable QR scale for larger or smaller codes  
- Works both **with background** or **plain QR**
